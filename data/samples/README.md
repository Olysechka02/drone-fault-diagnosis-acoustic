# Образцы аудио для проверки модели

Три файла из исходного датасета (Yi et al., 2023) — по одному на каждый из ключевых классов:

| Файл | Класс | Описание |
|---|---|---|
| `A_B_N_913_DuckPond_739_snr=13.54.wav` | N | Норма (исправный аппарат) |
| `A_B_MF1_185_DuckPond_637_snr=13.18.wav` | MF1 | Отказ мотора №1 |
| `A_B_PC1_867_DuckPond_599_snr=10.57.wav` | PC1 | Обрез пропеллера №1 |

**Структура имени файла:** `{model}_{maneuver}_{fault}_{index}_{background}_{bg_index}_snr={SNR}.wav`

- `A` — дрон Holy Stone HS720;
- `B` — манёвр Back (движение назад);
- `DuckPond` — фон с пруда (одна из 5 точек записи фонового шума);
- `snr=X` — отношение сигнал/шум в децибелах.

**Параметры:** 16 кГц, моно, 0.5 секунды, формат WAV.
